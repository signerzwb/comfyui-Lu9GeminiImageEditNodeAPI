节点简介
「ComfyUI 自定义 API 调用 Gemini 节点」是一款专为 ComfyUI 设计的灵活型图片编辑工具，基于 Google Gemini 2.5 Flash 模型的 API 能力，实现 “原始图片 + 文本指令” 驱动的智能编辑。节点深度适配 ComfyUI 的工作流逻辑，支持通过可视化参数配置完成接口调用，无需编写代码即可快速接入 Gemini 的强大图像生成与修改能力，同时保留极高的定制自由度，兼顾易用性与扩展性。
核心优势
1. 灵活可定制，适配多场景需求
接口 URL 完全可配置：支持自定义 API 地址、替换模型名称，适配不同环境下的接口部署（如私有部署、第三方代理接口等），无需修改节点代码即可切换调用目标；
编辑参数自由调节：图片比例（1:1/4:3/16:9/9:16）、分辨率（1080P/2K/4K）可按需选择，文本指令支持任意编辑需求（颜色修改、元素增减、场景替换、风格转换等），无固定功能限制。
2. 高通用性，打破场景与格式壁垒
输入输出兼容 ComfyUI 生态：无缝对接 ComfyUI 的 IMAGE 张量格式，可直接与 “Load Image”“Save Image”“ControlNet” 等节点串联，融入任意图像生成 / 处理工作流；
支持多样化编辑需求：无论是简单的颜色调整、背景替换，还是复杂的元素添加、风格迁移，均可通过自然语言指令实现，适用于设计优化、内容创作、素材加工等多场景；
跨格式自适应：自动完成 ComfyUI 张量与 Base64 格式的双向转换，兼容 PNG 图像的无损传输，无需手动处理格式转换问题。
3. 稳定可靠，降低使用门槛
内置网络容错机制：支持 3 次自动重试 + 120 秒超时配置，有效应对网络波动、接口响应延迟等问题，提升调用成功率；
友好的异常处理：接口调用失败时自动返回原始图片，避免工作流中断，同时在日志中输出详细错误信息，方便调试；
零代码上手：所有参数均通过 ComfyUI 可视化面板配置，无需掌握 API 调用、格式转换等技术细节，新手也能快速使用。
4. 扩展性强，适配未来需求
模型灵活切换：通过 “model_name” 参数可快速切换 Gemini 系列模型（如 gemini-pro、gemini-ultra 等），适配不同精度 / 速度需求；
支持二次开发：节点代码结构清晰，可轻松扩展新增参数（如请求头自定义、生成质量调节等），适配更多 API 协议与业务场景。
适用人群与场景
设计师 / 创作者：快速调整图片风格、替换元素，提升素材制作效率；
AI 绘画爱好者：在 ComfyUI 工作流中融入 Gemini 的图像编辑能力，丰富创作工具链；
开发者 / 研究者：快速对接 Gemini API，测试不同指令、模型参数的效果，无需搭建独立调用环境；
企业用户：适配私有部署的 Gemini API，满足内部合规需求，实现定制化图像处理流程。
使用价值
该节点以 “通用化、可定制” 为核心，打破了 API 调用与 ComfyUI 生态的壁垒，让 Gemini 的强大图像编辑能力成为 ComfyUI 工作流的 “模块化组件”。无需关注底层技术实现，只需聚焦核心需求，即可通过简单配置实现多样化的图像编辑效果，大幅提升工作流的灵活性与扩展性，是 ComfyUI 生态中一款兼具实用性与通用性的工具节点。
